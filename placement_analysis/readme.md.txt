# 📊 Placement Salary Prediction using Machine Learning

This project focuses on analyzing student placement data and building a machine learning model to predict salaries based on academic and placement-related features.

---

## 🎯 Objective

The objective of this project is to apply data preprocessing and machine learning techniques to develop a predictive model that estimates student salaries and demonstrates a complete data science workflow.

---

## 🚀 Key Features

* Data cleaning and preprocessing
* Handling missing values
* Encoding categorical variables
* Outlier detection and treatment
* Feature scaling
* Model building using Linear Regression
* Model evaluation

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn
* **Environment:** Jupyter Notebook

---

## 📂 Project Structure

```
placement-salary-prediction/
│
├── placement_analysis.ipynb   # Main notebook
├── README.md                  # Project documentation
```

---

## ⚙️ Workflow

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Feature Engineering
4. Handling Outliers
5. Feature Scaling
6. Model Training (Linear Regression)
7. Model Evaluation

---

## 📊 Results

The model is able to predict student salaries based on input features and demonstrates the effectiveness of basic regression techniques on structured data.
