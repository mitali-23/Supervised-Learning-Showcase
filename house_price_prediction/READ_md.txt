# 🏡 House Price Prediction using Linear Regression

## 📌 Overview

This project demonstrates a simple implementation of **Linear Regression** using Python to predict house prices based on area. It walks through data creation, visualization, model training, prediction, and evaluation.

The goal is to understand how machine learning models can be applied to real-world regression problems in a simple and intuitive way.

## 🚀 Steps Implemented

### 1. Import Libraries

Essential libraries for data manipulation, visualization, and modeling are imported.

### 2. Create Dataset

A dictionary is used to define house areas and their corresponding prices, which is converted into a Pandas DataFrame.

### 3. Data Visualization

A scatter plot is used to visualize the relationship between:

* Area (independent variable)
* Price (dependent variable)

### 4. Model Training

A **Linear Regression model** from Scikit-learn is used:

* Input: Area
* Output: Price

### 5. Regression Line

The best-fit line is plotted over the scatter plot to visualize the model's predictions.
